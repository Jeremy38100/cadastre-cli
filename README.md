__cadastre-cli__ is a tool to query french cadastre. From city (_nom_ ou _code postal_), _section_, and _numero de parcelle_ find:
 * surface area (m²)
 * location (lat, lng)
 * link to _GoogleMaps_ and _Geoportail_
